# homebrew-tap

Contains the Danger homebrew formulae for:

- Danger Swift
  ```sh
  brew install danger/tap/danger-swift
  ```
  
- Danger JS
  ```sh
  brew install danger/tap/danger-js 
  ```
  
- Danger Kotlin
  * ARM Chip (M1, M2)
    ```sh
    brew install danger/tap/danger-kotlin
    ```
  * Intel Chip
    ```sh
    brew install danger/tap/danger-kotlin-intel
    ```
